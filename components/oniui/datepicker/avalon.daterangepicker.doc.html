daterangepicker

Name 日期范围选择日历

daterangepicker是日期范围选择组件，可以通过设置起始日期与结束日期的最小间隔天数和最大间隔天数将截止日期限制在一定的选择范围中，同时通过点击确定按钮确定日期范围，在选择的时候日期范围框的下方会实时的说明当前选择情况

使用说明

名字 类型 默认值 说明
配置参数
fromLabel String '选择起始日期' 设置起始日期日历框的说明文字
toLabel String '选择结束日期' 设置结束日期日历框的说明文字
rules String "" 设置双日历框的工作规则 
            {
                rules: 'null, 0D, 8D',
                fromMinDate: '2014-05-02',
                fromMaxDate: '2014-06-28',
                toMinDate: '2014-06-01',
                toMaxDate: '2014-07-12'
            }
可以是绑定组件时定义的配置对象中的一个rules对象，也可以是一个字符串，指向一个上述对象。 其中对象中的rules属性定义结束初始日期异常时默认显示的日期、初始日期和结束日子之间最小相隔天数、最大相隔天数，格式是[+-]\d[DMY]，分别代表几天、几个月或者几年，也可以附加+或者-号，+号表示正数几天，-号表示负数几天 fromMinDate代表起始日期可以设置的最小日期 fromMaxDate代表起始日期可以设置的最大日期 toMinDate代表结束日期可以设置的最小日期 toMaxDate代表结束日期可以设置的最大日期
label String "" 模拟输入域的初始说明文字
defaultLabel String "日期范围" 选中日期之后，模拟输入域开始的说明文字
disabled Boolean false 设置是否禁用组件
separator String "-" 日期格式的分隔符，可以是"/"或者你希望的符号，但如果是除了"-"和"/"之外的字符则需要和parseDate和formatDate配合使用，以便组件能正常运作
startDay Number 1 设置每一周的第一天是哪天，0代表Sunday，1代表Monday，依次类推, 默认从周一开始
dateRangeWidth Number 260 配置日期范围框的宽度
shortcut Boolean false 是否在组件中显示日期选择快捷按钮
onOpen Function avalon.noop 打开daterangepicker的回调
参数名/返回值 类型 说明
vmodel Object 组件对应的Vmodel
onClose Function avalon.noop 关闭daterangepicker的回调
参数名/返回值 类型 说明
vmodel Object 组件对应的Vmodel
onSelect Function avalon.noop 点击日期范围框下方的确定按钮之后的回调
参数名/返回值 类型 说明
inputFromDate String 选择的起始日期
inputToDate String 选择的结束日期
oldValue Array 最近一次选中的起始和结束日期组成的数组
vmodel Object 组件对应的Vmodel
data Object 绑定组件元素上的data属性集合
parseDate(str) Function 将符合日期格式要求的字符串解析为date对象并返回，不符合格式的字符串返回null,用户可以根据自己需要自行配置解析过程
参数名/返回值 类型 说明
str String 需要解析的日期字符串
返回 Date 解析后的日期对象
formatDate(date) Function 将日期对象转换为符合要求的日期字符串
参数名/返回值 类型 说明
date Date 需要格式化的日期对象
返回 String 格式化后的日期字符串
datesDisplayFormat(label,fromDate,toDate) Function 配置日期范围框的显示格式
参数名/返回值 类型 说明
label String 日期范围提示文字
fromDate String 起始日期
toDate String 结束日期
返回 String 日期范围框要显示的内容
  • 基本配置的日期范围框，设置其初始说明文字、起始日历说明文字、结束日历说明文字
  • 配置日期范围框每一周第一天从周日开始、选择框下方说明文字显示内容
  • 初始化起始日期结束日期、各种方式切换禁用日历组件
  • 初始日期和截止日期之间的最小相隔天数和最大相隔天数、最小日期可选日期、最大可选日期
  • 自定义日期范围框解析和显示规则
  • 日期的onSelect回调、初始值为null的处理、setDates修正日期、setLabel修正提示文字
