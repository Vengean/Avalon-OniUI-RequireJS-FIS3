coupledatepicker demo

默认配置的双日历框
-

说明：双日历框的DOM结构为: 

    <div ms-widget="coupledatepicker">
        <div class="oni-coupledatepicker-item"></div>
        <span class="label">-</span>
        <div class="oni-coupledatepicker-item"></div>
    </div>

也就是绑定元素的子元素只能是起始日期包装div元素、分隔span元素、结束日期包装div元素

默认配置的双日历框

说明：双日历框除了上述的dom结构外也可以让其子元素为空，组件会自动创建需要的包装子元素

                <!DOCTYPE HTML>
                <html>
                <head>
                    <meta charset='utf-8'/>
                    <title>coupledatepicker demo</title>
                    <script src='avalon.js'></script>
                </head>
                <body>
                    <div class="demo-show" ms-controller="demo">
                        <fieldset style="position: relative; z-index: 2">
                            <legend>默认配置的双日历框</legend>
                            <div ms-widget="coupledatepicker">
                                <div class="oni-coupledatepicker-item"></div>
                                <span class="label">-</span>
                                <div class="oni-coupledatepicker-item"></div>
                            </div>
                            <p>说明：双日历框的DOM结构为:
                                <pre ms-skip>
                                    <div ms-widget="coupledatepicker">
                                        <div class="oni-coupledatepicker-item"></div>
                                        <span class="label">-</span>
                                        <div class="oni-coupledatepicker-item"></div>
                                    </div>
                                </pre>
                            </p>
                            <p>也就是绑定元素的子元素只能是起始日期包装div元素、分隔span元素、结束日期包装div元素</p>
                        </fieldset>
                        <fieldset style="position: relative; z-index: 1">
                            <legend>默认配置的双日历框</legend>
                            <div ms-widget="coupledatepicker"></div>
                            <p>说明：双日历框除了上述的dom结构外也可以让其子元素为空，组件会自动创建需要的包装子元素</p>
                        </fieldset>
                        <script>

                                var model = avalon.define("demo", function(vm) {
                                })

                            })
                        </script>
                    </div>
                </body>
