daterangepicker demo

日期范围框，startDay配置每周从周几开始

日历默认每周的第一天从周一开始，可以配置为从任一天开始，但是周日用0表示，周一到周六分别用1-6表示

日期范围框，设置日期选择框下方的选择说明文字

通过opts.msgFormat设置日期选择框下方日期选择说明文字内容形式

                <!DOCTYPE HTML>
                <html>
                <head>
                    <meta charset='utf-8'/>
                    <title>daterangepicker demo</title>
                    <script src='avalon.js'></script>
                </head>
                <body>
                    <div class="demo-show" ms-controller="demo">
                        <fieldset>
                            <legend>日期范围框，startDay配置每周从周几开始</legend>
                            <div ms-widget="daterangepicker" data-daterangepicker-start-day="0"></div>
                            <p>日历默认每周的第一天从周一开始，可以配置为从任一天开始，但是周日用0表示，周一到周六分别用1-6表示</p>
                        </fieldset>
                        <fieldset>
                            <legend>日期范围框，设置日期选择框下方的选择说明文字</legend>
                            <div ms-widget="daterangepicker, a, $aOpts"></div>
                            <p>通过opts.msgFormat设置日期选择框下方日期选择说明文字内容形式</p>
                        </fieldset>
                        <script> 
                            require(["datepicker/avalon.daterangepicker"], function(avalon) {
                                var model = avalon.define("demo", function(vm) {
                                    vm.$aOpts = {
                                        opts: {
                                            msgFormat: function(from, to) {
                                                return '已选时间段：' + from.getRawValue() + ' 至 ' + to.getRawValue() + ' 共计' + (Math.floor( (to.getDate().getTime() - from.getDate().getTime())/1000/60/60/24 + 1 ) - 1) + '晚';
                                            }
                                        }
                                    }
                                })
                                avalon.scan();
                            })
                        </script>
                    </div>
                </body>
                </html>
