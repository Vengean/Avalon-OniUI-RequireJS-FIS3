datepicker demo

切换日历的禁用与否

说明：初始禁用输入域，也就禁用了日历的显示，可以通过点击旁边的“切换禁用”按钮，使得日历可用

初始化日期，显示日历时高亮初始日期

说明：可以通过ms-duplex初始化日期输入域，也可以通过input的value属性直接设置，不过ms-duplex更灵活, 试着点击右边的“change date”看看会发生什么

手动输入日期使输入域失去焦点，看看date的变化

说明：手动输入日期然后使输入域失去焦点，看看date的变化。datepicker有自动校正的能力，也就是说当你的初始值不符合日期格式(包括无初始值的情况)而且allowBlank为默认的false时，datepicker会自动将初始值设置为今天

                        <script> 
                            require(["datepicker/avalon.datepicker"], function(avalon) {
                                var model = avalon.define("demo", function(vm) {
                                    vm.disabled = true;
                                    vm.toggleDisabled = function() {
                                        model.disabled = !model.disabled;
                                    }
                                    vm.date = "2014-06-02";
                                    vm.date1 = "2012-;"
                                    vm.changeDate = function() {
                                        var arr = avalon.range(10, 20);
                                        vm.date = "2014-06-"+arr[Math.floor(Math.random()*11)];
                                    }
