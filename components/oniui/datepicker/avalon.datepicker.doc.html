datepicker

日期选择器

datepicker组件方便快速创建功能齐备的日历组件，通过不同的配置日历可以满足显示多个月份、通过prev、next切换月份、或者通过下拉选择框切换日历的年份、月份，当然也可以手动输入日期，日历组件也会根据输入域中的日期值高亮显示对应日期等等各种需求

使用说明

名字 类型 默认值 说明
配置参数
calendarWidth Number 196 设置日历展示宽度
startDay Number 1 设置每一周的第一天是哪天，0代表Sunday，1代表Monday，依次类推, 默认从周一开始
minute Number 0 设置time的默认minute
hour Number 0 设置time的hour
width Number 90 设置日历框宽度
showTip Boolean true 是否显示节日提示
disabled Boolean false 是否禁用日历组件
changeMonthAndYear Boolean false 是否可以通过下拉框选择月份或者年份
mobileMonthAndYear Boolean false PC端可以通过设置changeMonthAndYear为true使用dropdown的形式选择年份或者月份，但是移动端只能通过设置mobileMonthAndYear为true来选择月份、年份
showOtherMonths Boolean false 是否显示非当前月的日期
numberOfMonths Number 1 一次显示的日历月份数, 默认一次显示一个
allowBlank Boolean false 是否允许日历框为空
minDate Object null 最小的可选日期，可以配置为Date对象，也可以是yyyy-mm-dd格式的字符串，或者当分隔符是“/”时，可以是yyyy/mm/dd格式的字符串
maxDate Object null 最大的可选日期，可以配置为Date对象，也可以是yyyy-mm-dd格式的字符串，或者当分隔符是“/”时，可以是yyyy/mm/dd格式的字符串
stepMonths Number 1 当点击next、prev链接时应该跳过几个月份, 默认一个月份
toggle Boolean false 设置日历的显示或者隐藏，false隐藏，true显示
separator String "-" 日期格式的分隔符,默认“-”，可以配置为"/"，而且默认日期格式必须是yyyy-mm-dd
calendarLabel String "选择日期" 日历组件的说明label
onChangeMonthYear Function avalon.noop 当month或者year更新时调用的回调
参数名/返回值 类型 说明
year Number 当前日期的year
month Number 当前日期的month(0-11)
vmodel Number 日历组件对应vmodel
dateCellRender Function false 格式化输出日期单元格内容
参数名/返回值 类型 说明
date Date 当前的日期
vmodel Vmodel 日历组件对应vmodel
dateItem Object 对应的包含日期相关信息的对象
watermark Boolean true 是否显示水印文字
zIndex Unary -1 设置日历的z-index
showDatepickerAlways Boolean false 是否总是显示datepicker
timer Boolean false 是否在组件中可选择时间
position String '' 设置datepicker的显示位置，可以为"rb"、"lt"、"rt"或者自定义的class,默认""
onBeforeRender Function null 在渲染日期前的准备工作，比如格式化日期前需请求的服务器数据
参数名/返回值 类型 说明
vmodel Object 当前日期组件对应的Vmodel
month Number 当前month(0-11)
year Number 当前year
onSelect Function avalon.noop 选中日期后的回调
参数名/返回值 类型 说明
date String 当前选中的日期
vmodel Object 当前日期组件对应的Vmodel
data Object 绑定组件的元素的data属性组成的集合
onClose Function avalon.noop 日历关闭的回调
参数名/返回值 类型 说明
date Object 当前日期
vmodel Object 当前日期组件对应的Vmodel
onSelectTime Function avalon.noop 在设置了timer为true时，选择日期、时间后的回调
参数名/返回值 类型 说明
vmodel Object 当前日期组件对应的Vmodel
parseDate Function parseDate 将符合日期格式要求的字符串解析为date对象并返回，不符合格式的字符串返回null,用户可以根据自己需要自行配置解析过程
参数名/返回值 类型 说明
str String 要解析的日期字符串
返回 Date Date格式的日期
formatDate(date) Function 将日期对象转换为符合要求的日期字符串
参数名/返回值 类型 说明
date Date 要格式化的日期对象
返回 String 格式化后的日期
