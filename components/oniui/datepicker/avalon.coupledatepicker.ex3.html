coupledatepicker demo

双日历框,初始化起始日期和结束日期

from is : {{from}}

to is : {{to}}

说明：通过配置duplex选项，初始化起始日期和结束日期，默认日期格式是"yyyy-mm-dd"，当没有初始值或者初始值格式错误时，组件会自动更正为今天所对应的日期

双日历框，禁用双日历框

说明：配置双日历组件的disabled为true来禁用组件，配置值可以是布尔值，也可以是controller下的一个属性变量名，如果是属性名则可以通过控制此属性值控制组件的禁用与否

双日历框，禁用双日历框

说明：配置disabled为一个变量名，点击右边的"切换禁用"按钮来切换双日历框的禁用与否

                <!DOCTYPE HTML>
                <html>
                <head>
                    <meta charset='utf-8'/>
                    <title>coupledatepicker demo</title>
                    <script src='avalon.js'></script>
                </head>
                <body>
                    <div class="demo-show" ms-controller="demo">
                        <fieldset style="position: relative; z-index: 3">
                            <legend>双日历框,初始化起始日期和结束日期</legend>
                            <div ms-widget="coupledatepicker" data-coupledatepicker-duplex="from, to"></div>
                            <p>说明：通过配置duplex选项，初始化起始日期和结束日期，默认日期格式是"yyyy-mm-dd"，当没有初始值或者初始值格式错误时，组件会自动更正为今天所对应的日期</p>
                        </fieldset>
                        <fieldset style="position: relative; z-index: 2">
                            <legend>双日历框，禁用双日历框</legend>
                            <div ms-widget="coupledatepicker" data-coupledatepicker-disabled="true"></div>
                            <p>说明：配置双日历组件的disabled为true来禁用组件，配置值可以是布尔值，也可以是controller下的一个属性变量名，如果是属性名则可以通过控制此属性值控制组件的禁用与否</p>
                        </fieldset>
                        <fieldset style="position: relative; z-index: 1">
                            <legend>双日历框，禁用双日历框</legend>
                            <div ms-widget="coupledatepicker, a"></div>
                            <p>说明：配置disabled为一个变量名，点击右边的"切换禁用"按钮来切换双日历框的禁用与否</p>
                            <button ms-click="toggleDisabled" style="position:absolute;right:0;top: 20px;">切换禁用</button>
                        </fieldset>
                        <script> 
                            require(["datepicker/avalon.coupledatepicker"], function(avalon) {
                                var model = avalon.define("demo", function(vm) {
                                    vm.from = "2014-06-03";
                                    vm.to = "2014-06-26";
                                    vm.toggleDisabled = function() {
                                        avalon.vmodels.a.disabled = !avalon.vmodels.a.disabled
                                    }
                                })
                                avalon.scan();
                            })
                        </script>
                    </div>
                </body>
                </html>
