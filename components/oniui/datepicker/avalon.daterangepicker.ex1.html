daterangepicker demo

默认配置项的日期范围选择框
设置日期范围框的起始日期和结束日期的日历说明

fromLabel设置起始日期的日历label，toLabel设置结束日期的日历label

设置日期范围框的说明文字

laebl设置日期范围框的默认说明文字，当选中日期时会自动更新此label

                <!DOCTYPE HTML>
                <html>
                <head>
                    <meta charset='utf-8'/>
                    <title>daterangepicker demo</title>
                    <script src='avalon.js'></script>
                </head>
                <body>
                    <div class="demo-show" ms-controller="demo">
                        <fieldset>
                            <legend>默认配置项的日期范围选择框</legend>
                            <div ms-widget="daterangepicker"></div>
                        </fieldset>
                        <fieldset>
                            <legend>设置日期范围框的起始日期和结束日期的日历说明</legend>
                            <div ms-widget="daterangepicker" data-daterangepicker-from-label="起始日期" data-daterangepicker-to-label="结束日期"></div>
                            <p>fromLabel设置起始日期的日历label，toLabel设置结束日期的日历label</p>
                        </fieldset>
                        <fieldset>
                            <legend>设置日期范围框的说明文字</legend>
                            <div ms-widget="daterangepicker" data-daterangepicker-label="请选择起始日期和结束日期"></div>
                            <p>laebl设置日期范围框的默认说明文字，当选中日期时会自动更新此label</p>
                        </fieldset>
                        <script> 
                            require(["datepicker/avalon.daterangepicker"], function(avalon) {
                                var model = avalon.define("demo", function(vm) {
                                    
                                })
                                avalon.scan();
                            })
                        </script>
                    </div> 
                </body>
                </html>
