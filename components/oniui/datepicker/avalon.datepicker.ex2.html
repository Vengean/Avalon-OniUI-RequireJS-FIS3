datepicker demo

使日历组件从周日开始显示

说明：配置startDay为0，使得日历从周日开始显示，当然也可以配置为2-6，默认是1

使日历可以通过下拉框切换月份和年份的显示

说明：配置changeMonthAndYear为true使得用户可以通过下拉选项切换月份和年份的显示

使日历可以通过下拉框切换月份和年份的显示

说明：根据minDate和maxDate配置可选的年份和月份

                <!DOCTYPE HTML>
                <html>
                <head>
                    <meta charset='utf-8'/>
                    <title>datepicker demo</title>
                    <script src='avalon.js'></script>
                </head>
                <body>
                    <div class="demo-show" ms-controller="demo">
                        <fieldset style="position: relative; z-index: 3">
                            <legend>使日历组件从周日开始显示</legend>
                            <input ms-widget="datepicker" data-datepicker-start-day="0" />
                            <p>说明：配置startDay为0，使得日历从周日开始显示，当然也可以配置为2-6，默认是1</p>
                        </fieldset>
                        <fieldset style="position: relative; z-index: 2">
                            <legend>使日历可以通过下拉框切换月份和年份的显示</legend>
                            <input ms-widget="datepicker" data-datepicker-change-month-and-year="true" />
                            <p>说明：配置changeMonthAndYear为true使得用户可以通过下拉选项切换月份和年份的显示</p>
                        </fieldset>
                        <fieldset style="position: relative; z-index: 1">
                            <legend>使日历可以通过下拉框切换月份和年份的显示</legend>
                            <input ms-widget="datepicker" data-datepicker-change-month-and-year="true" data-datepicker-min-date="2012-01-01" data-datepicker-max-date="2015-12-31"/>
                            <p>说明：根据minDate和maxDate配置可选的年份和月份</p>
                        </fieldset>
                        <script> 
                            require(["datepicker/avalon.datepicker"], function(avalon) {
                                var model = avalon.define("demo", function(vm) {
                                    vm.$aOpts = {
                                        dayNames: ["Sun", "Mon", "Tue", "Wed", "Thu", "Fri", "Sat"]
                                    }
                                })
                                avalon.scan();
                            })
                        </script>
                    </div>
                </body>
                </html>
