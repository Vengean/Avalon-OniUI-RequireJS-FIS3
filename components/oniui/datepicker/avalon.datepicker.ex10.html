datepicker demo

移动端不使用dropdown切换月份或年份

说明：设置mobileMonthAndYear为true使得datepicker在mobile下通过mobile的形式选择月份和年份而不是dropdown的形式

移动端不使用dropdown切换月份或年份

说明：根据minDate和maxDate确定可选择的年份和月份

                <!DOCTYPE HTML>
                <html>
                <head>
                    <meta charset='utf-8'/>
                    <title>datepicker demo</title>
                    <script src='avalon.js'></script>
                </head>
                <body>
                    <div class="demo-show" ms-controller="demo">
                        <fieldset style="position:relative; z-index: 2">
                            <legend>移动端不使用dropdown切换月份或年份</legend>
                            <input ms-widget="datepicker" data-datepicker-mobile-month-and-year="true"/>
                            <p>说明：设置mobileMonthAndYear为true使得datepicker在mobile下通过mobile的形式选择月份和年份而不是dropdown的形式</p>
                        </fieldset>
                        <fieldset style="position:relative; z-index: 1">
                            <legend>移动端不使用dropdown切换月份或年份</legend>
                            <input ms-widget="datepicker" data-datepicker-min-date="2012-01-01" data-datepicker-max-date="2015-12-31" data-datepicker-mobile-month-and-year="true"/>
                            <p>说明：根据minDate和maxDate确定可选择的年份和月份</p>
                        </fieldset>
                        <script> 
                            require(["datepicker/avalon.datepicker"], function(avalon) {
                                var model = avalon.define("demo", function(vm) {
                                })
                                avalon.scan();
                            })
                        </script>
                    </div>
                </body>
                </html>
