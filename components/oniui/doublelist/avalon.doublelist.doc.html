doublelist

doublelist组件

以左右列表形式展示实现的复选组件，不支持ms-duplex，请在onChange回调里面处理类似ms-duplex逻辑

使用说明

名字 类型 默认值 说明
配置参数
data Array [] 配置左侧待选项列表，数据 [{value: xxx, name: xx}]
select Array [] 选中的value list，[value1,value2]，取的是data 里面item的value
toggle Boolean true 组件是否显示，可以通过设置为false来隐藏组件
hideSelect Boolean false 是否隐藏以选中的项目，默认不隐藏
onInit Member avalon.noop onInit(vmodel, options, vmodels) 完成初始化之后的回调,call as element's method
getTemplate(tmpl,opts,tplName) Function getTemplate(tpl, opts) 定制修改模板接口
countLimit(select) Function countLimit(select) 选择条目限制，必须有return true or false，参数是当前已选中条数和add or delete操作
onChange Member avalon.noop change(newValue, oldValue, vmodel) 所选变化的回调，不建议使用，等价于onChange
接口方法与固有属性
reset(data,select) Function reset(data, select) 重置，用新的data和select渲染，如果!data为真，则不修改左侧list；如果select为空或者空数组，则清空已选，否则将select中的项目置为已选
  • doublelist-隐藏已选中条目，限制选中条数
  • doublelist-显示已选中条目，不限制选中条数
