rating

评分组件

使用ms-duplex可以方便的设置和获取星标数值。

图标使用字体图标。来自： Font-Awesome

使用说明

名字 默认值 说明
配置参数
defaultValue 3 设置星标默认值
margin 3 设置星标间距
max 5 设置星标最大值
notSelectedColor #CECECE 设置未选中颜色
notSelectedContent &#xf006; 设置未选中图标
selectedColor #00A3C2 设置选中颜色
selectedContent &#xf005; 设置选中图标
size 20 设置星标大小
getTemplate(tmp) Function 方便用户自定义模板
onInit(vmodel, options, vmodels) Function 组件初始化完成回调
接口方法
set(num) 设置星标值。接受一个自然数为参数。
getRating() 获取当前星标值
注意

字体编码获取方式：

星星图标详情地址 http://fortawesome.github.io/Font-Awesome/icon/star/，如下图找到对应编码为f005

则此图标编码为:&#xf005;
