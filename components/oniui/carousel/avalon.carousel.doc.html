carousel

Name 图片轮播组件图片轮播，采用跑马灯效果

使用说明

名字 类型 默认值 说明
配置参数
pictures Array [] 轮播图片素材，每个图片可以用对象方式配置src,alt,href,title,description
pictureWidth Number 500 组件宽度
pictureHeight Number 200 组件高度
effect String "slide" 图片切换类型，none:无特效 / fade:渐隐 / slide:滑动
easing String "easeInOut" 缓动类型， linear:无缓动效果 / easeIn:在过渡的开始提供缓动效果 / easeOut:在过渡的结尾提供缓动效果 / easeInOut 在过渡的开始和结尾提供缓动效果
timeout Number 2500 图片切换时间间隔
duration Number 300 图片切换速度
showDescription Boolean false 显示图片描述
alwaysShowArrow Boolean true 显示左右切换箭头
alwaysShowSelection Boolean true 显示底部圆形切换部件
autoSlide Boolean true 自动播放
hoverStop Boolean true 鼠标经过停止播放
eventType String "click" 触发导航切换图片的事件类型，click\mouseenter\both
arrowLeftNormalSrc String "default.png" 左箭头正常状态图标
arrowRightNormalSrc String "default.png" 右箭头正常状态图标
arrowLeftHoverSrc String "default.png" 左箭头hover状态图标
arrowRightHoverSrc String "default.png" 右箭头hover状态图标
lazyload Boolean true 图片懒加载
lazyloadImg String "default.png" 懒加载loading图

  • 图片轮播组件-自定义宽高
  • 图片轮播组件-自定义图片切换时间间隔 / 图片切换速度
  • 图片轮播组件-配置小部件
  • 图片轮播组件-自定义切换动画效果
  • 图片轮播组件-自定义动画缓动类型
  • 图片轮播组件-自适应容器尺寸
  • 图片轮播组件-自定义图片标题和描述
  • 图片轮播组件-自定义懒加载loading图
