最基本的dialog组件（ms-widget="dialog dialogId"）

说明：配置ms-widget="dialog,dialogId"就可实现一个简单的弹窗组件，组件默认有确定和取消按钮，点击任意按钮都会关闭弹窗，当然弹窗右上角也会显示关闭按钮，点击关闭弹窗，dialog的子元素可以是模板，组件会自动渲染

                <!DOCTYPE HTML>
                <html>
                <head>
                    <meta charset='utf-8'/>
                    <title>最基本的dialog组件</title>
                    <script src='avalon.js'></script>
                </head>
                <body>
                    <div class="demo-show" ms-important="demo">
                        <fieldset>  
                            <legend>基本的dialog组件</legend>
                                <button ms-click="show('aa')">打开对话框</button>
                                <div ms-widget="dialog,aa">
                                    呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵呵
                                    <p>{{message}}</p>
                                    <ul ms-widget="checkboxlist" data-checkboxlist-duplex="check1">
                                        <li><input type="checkbox" value="1">选项一</li>
                                        <li><input type="checkbox" value="2">选项二</li>
                                        <li><input type="checkbox" value="3">选项三</li>
                                    </ul>
                                </div>
                        </fieldset>
                        <script> 
                            require(["dialog/avalon.dialog", "checkboxlist/avalon.checkboxlist"], function() {
                                avalon.define("demo", function(vm) {
                                    vm.message = "笑傻了吧！！";
                                    vm.show = function( id ){
                                        avalon.vmodels[id].toggle = true;
                                    }
                                    vm.check1 = ["1"];
                                })
                                avalon.scan();
                            })
                        </script>
                    </div>
                </body>
                </html>
