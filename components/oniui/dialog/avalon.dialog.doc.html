dialog

Name 对话框

dialog组件提供弹窗显示或者隐藏,通过简单配置可以水平居中显示dialog弹窗，此组件支持弹窗中再弹窗，也可以用来模拟alert的行为，非常方便

使用说明

名字 类型 默认值 说明
配置参数
setContent(content,noScan,contentVmodels) Function 可以动态改变dialog的显示内容
参数名/返回值 类型 说明
content String 要替换的content，可以是已经渲染ok的view也可以是未解析渲染的模板
noScan Boolean 当content是模板时noScan设为false或者不设置，组件会自动解析渲染模板，如果是已经渲染ok的，将noScan设为true，组件将不再进行解析操作
contentVmodels Array 如果content为未解析的模板，noScan为false，contentVmodels是用来解析模板content的vmodels
width Number 480 设置dialog的width
title String " " 设置弹窗的标题
draggable Boolean false 设置dialog是否可拖动
type String "confirm" 配置弹窗的类型，可以配置为alert来模拟浏览器
content String "" 配置dialog的content，默认取dialog的innerHTML作为dialog的content，如果innerHTML为空，再去取content配置项.需要注意的是：content只在初始化配置的起作用，之后需要通过setContent来动态的修改
onConfirm Function avalon.noop 定义点击"确定"按钮后的回调操作
参数名/返回值 类型 说明
event Number 事件对象
vmodel Object 组件对应的Vmodel
返回 Boolean 如果return false，dialog不会关闭，用于异步操作
onOpen Function avalon.noop 定义显示dialog时的回调
参数名/返回值 类型 说明
vmodel Object 组件对应的Vmodel
onCancel Function avalon.noop 定义点击"取消"按钮后的回调操作
参数名/返回值 类型 说明
event Object 事件对象
vmodel Object 组件对应的Vmodel
返回 Boolean 如果return false，dialog不会关闭，用于异步操作
onClose Function avalon.noop 定义点击"关闭"按钮后的回调操作
参数名/返回值 类型 说明
event Object 事件对象
vmodel Object 组件对应的Vmodel
setTitle Member avalon.noop 动态修改dialog的title,也可通过dialogVM.title直接修改
setModel Function avalon.noop 重新渲染模板
参数名/返回值 类型 说明
m Object 重新渲染dialog的配置对象，包括title、content、content中涉及的插值表达式，需要注意的是，title和content不是真正渲染的内容，所以不需要avalon进行扫描监控，定义的时候必须在其前面加上"$",否则组件不会渲染成想要的效果
showClose Boolean true 配置dialog是否显示"取消"按钮，但是如果type为alert，不论showClose为true or false都不会显示"取消"按钮
toggle Boolean false 通过此属性的决定dialog的显示或者隐藏状态
widgetElement String "" 保存对绑定元素的引用
container String "body" dialog元素的上下文父元素，container必须是dialog要appendTo的父元素的id或者元素dom对象
confirmName String "确定" 配置dialog的"确定"按钮的显示文字
cancelName String "取消" 配置dialog的"取消"按钮的显示文字
getFooter(tmp) Function 通过此方法配置dialog的footer
参数名/返回值 类型 说明
tmp String dialog默认模板的footer
返回 String 用户自定义的dialog的footer
modal Boolean true 是否显示遮罩
zIndex Identifier maxZIndex 通过设置vmodel的zIndex来改变dialog的z-index,默认是body直接子元素中的最大z-index值，如果都没有设置就默认的为10
zIndexIncrementGlobal Number 0 相对于zIndex的增量, 用于全局配置，如果只是作用于单个dialog那么zIndex的配置已足够，设置全局需要通过avalon.ui.dialog.defaults.zIndexIncrementGlobal = Number来设置
