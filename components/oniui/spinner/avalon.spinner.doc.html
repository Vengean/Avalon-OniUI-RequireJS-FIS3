spinner

Name 数字输入框

spinner组件是用来增强输入框的能力，使其可以通过上下按钮或者键盘上的up、down键来改变输入域的数值，而且确保输入域始终是数值字符，非数值字符无效，组件会默认将非数值字符转换为最近一次的输入域数值

使用说明

名字 类型 默认值 说明
配置参数
min Identifier NaN spinner的最小值,默认不存在最小值限制
max Identifier NaN spinner的最大值，默认不存在最大值限制
step Number 1 spinner的步长
width String "auto" 设置spinner的width
value Number 0 spinner的当前值
disabled Boolean false 是否禁用spinner
onDecrease Function avalon.noop 减值更新spinner之后的回调
参数名/返回值 类型 说明
value Number 当前值
onIncrease Function avalon.noop 增值更新spinner之后的回调
参数名/返回值 类型 说明
value Number 当前值

spinner使用注意点

  1. 请保证输入域的值是数值或者字符串型的数值，否则会报错
  2. 当通过手动输入值时，如果输入的值不是数值型的，组件会默认将其置为修改之前的值，如果输入大于最大值，组件会自动将其置为最大值，小于最小值时同理
  3. 在输入域获得焦点时可通过键盘上的上下箭头控制spinner的增减
  4. spinner会判断初始输入域值是否在用户设置的数值范围呢，不在的话会进行调整
