draggable

拖拽组件让一个定位元素可以拖动。使用时在目标元素上添加ms-draggable绑定属性，值可以为空。值不为空时，有两个可选设置项，用逗号隔开。 第一个用于指放置配置对象的VM的$id，为$或不存在时默认为离它最近的VM。第二个用于指定配置对象。

使用说明

名字 默认值 说明
配置参数
axis "xy" 决定拖动方向， x是水平， y是垂直，xy是任意，此外的其他值都会转换为xy
ghosting false 影子拖动，指拖动时不拖动原元素，而是动态生成一个与原元素相同大小的半透明元素做替身，在拖动结束再决位到替身的位置上。 用于拖动复杂的目标节省性能
handle 空函数 手柄拖动，用于拖动原元素的某一部分就能拖动整体，因此这函数要求返回一个元素节点，此元素节点应该位于拖动元素内部。此函数有两个参数 分别是事件对象与data对象，this指向拖动元素
containment 区域拖动，可以是window, parent, #id, [0, 0, 400, 300]这些格式。 window是指在当前可视区拖动， parent是指当前父元素， #id是用于取得目标元素， 拥有4个数字的数组是用于圈定范围，前两个数字是左上角的坐标，后两个是右下角的坐标
beforeStart(e, data) 空函数 鼠标按下时立即触发的回调，它的参数与this参考handle。你可以在这里设置拖动范围与改变元素样式。
start(e, data) 空函数 紧接着beforeStart，handle触发的回调，它的参数与this参考handle
drag(e, data) 空函数 鼠标拖动（mousemove或touchmove）连续触发的回调，它的参数与this参考handle
beforeStop(e, data) 空函数 鼠标弹起后立即触发的回调，它的参数与this参考handle。你可以在恢复样式，或改变data的一些属性
stop(e, data) 空函数 拖放结束后执行的回调，它的参数与this参考handle。

使用说明

名称 默认值 说明
★element 要拖动的元素节点
$element 用avalon封装好element --> avalon(element)
★clone 影子拖动时，复制生成的元素节点
started false 是否已经在拖动
★pageX/pageY 0 当前元素在页面的坐标（这两个会在mousemove中改变）
startPageX/startPageY 0 开始拖动时元素在页面的坐标
★dragX/dragY true 是否在鼠标拖动时，将pageX/pageY赋给元素的left, top来改变它的位置
clickX/clickY 0 鼠标点击的位置与目标元素左上角的距离
startLeft/startTop 0 拖动前相对于offsetParent的坐标
endLeft/endTop 0 拖动后相对于offsetParent的坐标
endX/endY 0 拖动后相对于offsetParent的坐标
marginLeft/marginTop 0 开始拖动时元素的左上边距宽度
