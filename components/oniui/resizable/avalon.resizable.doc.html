resizable

伸缩组件使用时在目标元素上添加ms-resizable绑定属性，值可以为空。值不为空时，有两个可选设置项，用逗号隔开。 第一个用于指放置配置对象的VM的$id，为$或不存在时默认为离它最近的VM。第二个用于指定配置对象。

使用说明

名字 默认值 说明
配置参数
handles "n,e,s,w,ne,se,sw,nw" 伸缩的方向,一共八个方向
maxHeight 10000 被调整大小的元素最大能拉到多高
maxWidth 10000 被调整大小的元素最大能拉到多宽
minHeight 10 被调整大小的元素最小能压到多低
minWidth 10 被调整大小的元素最小能压到多窄
edge 5 距离在拉到的边缘还有多少ms就能被调整大小及改变光标的类型
aspectRatio 等比例缩放, 没有默认值,如果是数字,为长与高之比
start(e,data) 空函数 调整大小前触发的函数。此函数有两个参数 分别是事件对象与data对象，this指向拖动元素
resize(e,data) 空函数 调整大小时连续触发的函数。参数与this情况见start
drag(e,data) 空函数 拖动时连续触发的函数。参数与this情况见start。这个需要自己设置元素top, left。
stop 空函数 调整大小后触发的函数。参数与this情况见start

使用说明

名称 默认值 说明
接口方法与固有属性
startResizeTop/startResizeLeft 0 被调整大小的元素最开始时的坐标
startResizeWidth/startResizeHeight 0 被调整大小的元素最开始时的宽高
resizeTop/resizeLeft 0 被调整大小的元素在拖动中时的坐标
resizeWidth/resizeHeight 0 被调整大小的元素在拖动中时的宽高
dir "" 被调整大小的元素当前往哪个方向拉伸,为handles的其中一个
  • 各种回调
  • 最小最大尺寸
  • 按比例缩放
  • 使用shift键
