touch

touch组件

支持对手机浏览器下touch事件、手势

tap 拍击
doubletap 双击
hold 长按
swipe 单指滑动
swipeleft 单指向左滑动 - 必定出发swipe
swiperight 单指向右滑动 - 必定出发swipe
swipeup 单指向上滑动 - 必定出发swipe
swipedown 单指向下滑动 - 必定出发swipe 
avalon.bind(ele, "swipe*", function(e) {
    e.detail = {
        direction: left || right || up || down
    }
})
pinch 双指缩放
pinchin 双指捏合 - 必定出发pinch
pinchout 双指展开 - 必定出发pinch 
avalon.bind(ele, "pinch*", function(e) {
    e.detail = {
        direction: in || out,
        scale: 缩放比
    }
})
squeeze 三指以上指捏合
splay 三指以上展开
rotate 双指旋转
rotatecw 顺时针旋转 - 必定出发rotate
rotateccw 逆时针旋转 - 必定出发rotate 
avalon.bind(ele, "rotate*", function(e) {
    e.detail = {
        direction: cw || ccw,
        deflection: 旋转角度数， 单位一度
    }
})

支持对手机浏览器下shake事件绑定的支持

avalon.bind(ele, "shake", function(e) {
    e.direction = left || right || forward || back || up || down || 以及组合， 但不准确
})

        </ul>
    </div>
</div>当没有找到直系ms-repeat元素的时候，则会创建普通的iscroll

    avalon.define("test", function(vm) {
    vm.data = []  // 成对出现才行
    vm.data$ = [] // 这个必须有，跟repeat或者each对象成对出现
    vm.$options = {
    lineHeight: 25, // repeat元素的行高
    showLines: 10, // 显示的数量,
    getData: avalon.noop // 获取数据的函数
    }
    })
</script>*
