camera

图片百叶窗效果组件基于carousel图片轮播组件。注意：此组件动画效果由jquery完成，需要从外部引入。具体引入方法请看示例代码。

使用说明

名字 类型 默认值 说明
配置参数
pictures Array [] 轮播图片素材
effect String "random" 图片切换类型，取值：random:全部随机 / none:无特效 / slideX:横向滑动 / slideY:纵向滑动 / fadeIn:渐入 / crossX:横向交叉 / crossY:纵向交叉 / stepX:横向阶梯 / stepY:纵向阶梯 / rotateFadeIn:旋转式渐入
timeout Number 2500 切换时间间隔
speed Number 600 切换速度，数值越小速度越快，单位为毫秒
alwaysShowArrow Boolean true 显示左右切换箭头
alwaysShowSelection Boolean true 显示底部圆形切换部件
hoverStop Boolean true 鼠标经过停止播放
adaptiveWidth Boolean false 适应外围宽度
adaptiveHeight Boolean false 适应外围高度
eventType String "click" 触发tab切换的nav上的事件类型，取值click\mouseenter\both
arrowLeftClass String "" 左右箭头的className，可不传
arrowRightClass String "" 左右箭头的className，可不传
slicedCols Number 5 区块列数
slicedRows Number 4 区块行数
  • 图片百叶窗切换组件
  • 图片百叶窗切换组件-自定义宽高
  • 图片百叶窗切换组件-自定义切换间隔和切换速度
  • 图片百叶窗切换组件-自定义区块数量
