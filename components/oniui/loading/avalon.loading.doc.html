loading

加载效果组件

实现各种加载动画效果

使用说明

名字 类型 默认值 说明
配置参数
count Number 10 type=ball，loading效果组成的小图形个数
interval Number 120 type=ball，毫秒数，动画效果帧间隔
circleMargin Number 1 type=ticks，小球之间的间距，单位是一倍小球半径
count Number 3 type=ticks，小球个数
height Number 20 type=ticks，高度
interval Number 360 type=ticks，毫秒数，动画效果帧间隔
angel Number 90 type=spin，转动的弧形的角度，单位是1度
opacity Number 0.2 type=spin，背景圆弧的透明度
startangle Number 0 type=spin，圆弧开始的角度，单位1度
interval Number 36 type=spin，毫秒数，动画效果帧间隔
count Number 8 type=spokes，长方形个数
width Number 32 type=spokes，效果宽度,
spokesWidth Number 4 type=spokes，小长方形宽度
spokesHeight Number 8 type=spokes，小长方形高度
interval Number 125 type=spokes，效果动画间隔毫秒数
width Number 64 type=spinning-bubbles 宽度，小球的个数继承自type=ball
widthInner Number 54 type=spinning-bubbles 内宽
height Number 30 type=bubbles 高度，宽度继承type=spinning-bubbles
widthInner Number 50 type=bubbles 内宽
count Number 3 type=bubbles 球的个数
interval Number 360 type=bubbles 动画ms数
interval Member _config["ball"].interval type=spinning-spin 帧间隔，继承ball
count Number 8 type=spinning-spin 小圆弧个数，一般请保证 360 / 8 % padding = 0
width Number 46 type=spinning-spin 圆外直径
widthInner Number 38 type=spinning-spin 圆内直径
src String "https://source.qunarzz.com/piao/images/loading_camel.gif" type=img，loading效果的gif图片
width Number 52 type=img，loading效果宽度
height Number 39 type=img，loading效果高度
onInit Member avalon.noop onInit(vmodel, options, vmodels) 完成初始化之后的回调,call as element's method
color String "#619FE8" 效果的颜色
type String "ball" 类型，默认是ball，球，可取spin,ticks
toggle Boolean true 是否显示
modal Boolean true 是否显示遮罩
modalOpacity Number 0.1 遮罩透明度
modalBackground String "#fff" 遮罩背景色
container Unary void 0 loading效果显示的容器，默认是绑定widget的元素
getTemplate(tmpl,opts,tplName) Function getTemplate(tpl, opts, tplName) 定制修改模板接口
接口方法与固有属性
showLoading() Function showLoading() 显示loading效果
hideLoading() Function hideLoading() 隐藏loading
destroyLoading() Function destroyLoading() 销毁loading
appendTo(container) Function 将loading效果插入到指定的容器里
参数名/返回值 类型 说明
container 目标容器元素，默认是绑定widget的元素
  • loading:ball
  • loading:spin
  • loading:ticks
  • loading:spokes
  • loading:spinning-bubbles
  • loading:bubbles-这个只有在高级非IE浏览器上才能work good
  • loading:spinning-spin
  • loading:img
