cookie

Cookie工具模块

处理cookie的工具函数集合， 与store模块的接口一致， 即包含get, set, forEach, remove, clear, getAll

使用说明

名字 类型 默认值 说明
接口方法与固有属性
stringify(name,val,opts) Function 将两个字符串变成一个cookie字段 
Cookie.stringify('foo', 'bar', { httpOnly: true })  => "foo=bar; httpOnly"
参数名/返回值 类型 说明
name String cookie的名字不能为空
val String cookie的名字不能为空
opts Undefined|Object|Number 配置对象，如果为数字则当成maxAge,否则为对象时，里面可以配置maxAge, domain, path, expires, httpOnly, secure
forEach(callback) Function 遍历所有cookie
参数名/返回值 类型 说明
callback Function 里面会依次传入key与value
get(name) Function 获取某一cookie
参数名/返回值 类型 说明
name String
返回 String
getAll() Function 获取所有cookie，以对象形式返回
参数名/返回值 类型 说明
返回 Object
set(key,val,opts) Function 添加或设置某一cookie
参数名/返回值 类型 说明
name String
value String
返回 Undefined|Object|Number
remove(key,opt) Function 移除某一cookie
参数名/返回值 类型 说明
name String
opt Object|Undefined
clear() Function 移除所有cookie
