textbox demo 2

拥有占位符的textbox组件（ms-widget="textbox" placeholder="表单域获得焦点且表单域值为空时显示占位符，其他时候隐藏占位符"）

说明：表单域获得焦点且表单域值为空时显示占位符，其他时候隐藏占位符

                <!DOCTYPE HTML>
                <html>
                <head>
                    <meta charset='utf-8'/>
                    <title>拥有占位符的textbox组件</title>
                    <script src='avalon.js'></script>
                </head>
                <body>
                    <div class="demo-show" ms-controller="demo2">
                        <form>
                            <table style="width:600px; border:1px solid #000;border-collapse: collapse">
                                <tbody>
                                    <tr>
                                        <td style="width: 260px; border: 1px solid #000; padding: 20px">
                                            <textarea ms-widget="textbox" ms-duplex="areaempty"  type="text" placeholder="表单域为空且存在占位符表单域为空且存在占位符表单域为空且存在占位符表单域为空且存在占位符"></textarea>
                                            <p>输入框的值是： {{areaempty}}</p>

                                        </td>
                                        <td style="border: 1px solid #000; padding: 20px">
                                            <textarea ms-widget="textbox" ms-duplex="areavalue" type="text" placeholder="表单域不为空且存在占位符"></textarea>
                                            <p>输入框的值是： {{areavalue}}</p>
                                        </td>
                                    </tr>
                                </tbody>
                            </table>
                        </form>
                    </div>
                    <script>
                        require(["textbox/avalon.textbox"], function() {
                            avalon.define("demo2", function(vm) {
                                vm.areaempty = "";
                                vm.areavalue = "表单域存在占位符但是有值的情况";
                            })
                            avalon.scan();
                        })
                    </script>
                </body>
                </html>

输入框的值是： {{areaempty}}

输入框的值是： {{areavalue}}
