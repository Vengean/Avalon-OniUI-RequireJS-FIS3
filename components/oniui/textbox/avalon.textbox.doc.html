textbox

Name 具有提示功能的输入框

通过给简单的表单输入域设置不同的配置项可以使表单拥有舒服的视觉效果，也可以使其具有提示补全功能

使用说明

名字 类型 默认值 说明
配置参数
suggest Boolean false 配置输入框有自动提示补全功能，提示类型由用户自定义，默认配置为false，也就是不开启自动补全功能
suggestDisableLetter Boolean false 配置输入框进行自动提示补全时，是否对字母进行过滤，默认配置为false,也就是说输入值包含字母时也会进行自动补全
autoTrim Boolean true 是否自动过滤用户输入的内容头部和尾部的空格
tabIndex Unary -1 配置textbox在进行tab切换时的tabIndex，切换顺序从值小的开始，必须配置为大于0的值
width Unary -1 配置textbox的显示宽
adaptiveHeight Object "" 使textbox自适应高度，并可以配置自适应时的minHeight/maxHeight
autoFocus Boolean false 如果想要鼠标悬停在textbox上时就focus textbox，设置此属性为true即可
disabledClass String "oni-textbox-disabled" 配置输入域disabled时组件包装元素设置的类，多个类以空格分隔
getTemplate(tmp) Function 模板函数,方便用户自定义模板
参数名/返回值 类型 说明
str String 默认模板
opts Object vmodel
返回 String 新模板
stateClass String "" 为textbox添加样式，默认可以添加oni-textbox-error
suggestOnChange String "" 配置提示补全时切换提示项之后的callback
suggestFocus Boolean false 特殊的suggest，当focus时即显示特定的提示列表
接口方法与固有属性
widgetElement String "" 绑定组件元素的dom对象的引用

  • 拥有占位符的textbox
  • 切换禁用textbox
  • 有自动补全功能的textbox
  • 无视用户输入的自动补全
  • 添加回调操作的自动补全
