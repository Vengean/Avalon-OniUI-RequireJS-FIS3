mask

输入引导模块

这是ms-duplex2.0的一个扩展模块，用于引导用户输入。

通过如下方式使用:

<input ms-duplex-mask="a" data-duplex-mask="((00/00其他字符0000)" />
名字 类型 默认值 说明
配置参数
element Element element 组件实例要作用的input元素
dataMask String dataMask 用户在input/textarea元素上通过data-duplex-mask定义的属性值
placehoder Boolean "_" "_", 将元字符串换为"_"显示到element.value上，如99/99/9999会替换为__/__/____，可以通过data-duplex-mask-placehoder设置
hideIfInvalid Boolean false false, 如果它不匹配就会在失去焦点时清空value(匹配是指所有占位符都被正确的字符填上)，可以通过data-duplex-mask-hide-if-invalid设置
hideIfPristine Boolean true true如果它没有改动过就会在失去焦点时清空value，可以通过data-duplex-mask-hide-if-pristine设置
showIfHover Boolean false false 当鼠标掠过其元素上方就显示它出来，可以通过data-duplex-mask-show-if-hover设置
showIfFocus Boolean true true 当用户让其元素得到焦点就显示它出来，可以通过data-duplex-mask-show-if-focus设置
showAlways Boolean false false 总是显示它，可以通过data-duplex-mask-show-always设置
translations Object 此对象上每个键名都是元字符，都对应一个对象，上面有pattern(正则)，placehoder(占位符，如果你不想用"_")
元字符 意义
9 表示任何数字，0-9，正则为/\d/
A 表示任何字母，，正则为/[a-zA-Z]/
* 表示任何非空字符，正则为/\S/
