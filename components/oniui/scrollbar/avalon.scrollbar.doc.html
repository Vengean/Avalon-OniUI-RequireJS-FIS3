scrollbar

滚动条组件

自定义滚动条样式，绑定ms-widget="scrollbar"的元素内必须包含一个class="oni-scrollbar-scroller"的视窗元素

使用说明

名字 类型 默认值 说明
配置参数
disabled Boolean false 组件是否被禁用，默认为否
toggle Boolean true 组件是否显示，可以通过设置为false来隐藏组件
position String "right" scrollbar出现的位置,right右侧，bottom下侧，可能同时出现多个方向滚动条
limitRateV Number 1.5 竖直方向，拖动头最小高度和拖动头宽度比率
limitRateH Number 1.5 水平方向，拖动头最小宽度和高度的比率
scrollTop Number 0 竖直方向滚动初始值，负数会被当成0，设置一个极大值等价于将拖动头置于bottom
scrollLeft Number 0 水平方向滚动初始值，负数会被当成0处理，极大值等价于拖动头置于right
show String "always" never一直不可见，scrolling滚动和hover时候可见，always一直可见
showBarHeader Boolean true 是否显示滚动条两端的上下箭头
draggerHTML String "" 滚动条拖动头里，注入的html碎片
breakOutCallback Boolean false breakOutCallback(["h", "up"], vmodel) 滚动到极限位置的回调，用来实现无线下拉等效果 breakOutCallback(["h", "up"], vmodel) 第一个参数是一个数组，分别是滚动条方向【h水平，v竖直】和超出极限的方向【up是向上或者向左，down是向右或者向下】，第三个参数是一个对象，包含滚动条的元素，宽高等信息
onInit Member avalon.noop onInit(vmodel, options, vmodels) 完成初始化之后的回调,call as element's method
viewHeightGetter(viewElement) Function viewHeightGetter(viewElement) 配置计算视窗高度计函数，默认返回innerHeight
viewWidthGetter(viewElement) Function viewWidthGetter(viewElement) 配置计算视窗宽度计函数，默认返回innerWidth
getTemplate(tmpl,opts) Function getTemplate(tpl, opts) 定制修改模板接口
onScroll(newValue,oldValue,diretion,vmodel) Function onScroll(newValue, oldValue, diretion, vmodel) 滚动回调,scrollLeft or scrollTop变化的时候触发，参数为newValue, oldValue, diretion, vmodel diretion = h 水平方向，= v 竖直方向
size String "normal" srollbar size,normal为10px，small为8px，large为14px
接口方法与固有属性
getBars() Function getBars()返回所有的滚动条元素，avalon元素对象
getScroller() Function getScroller()返回scroller avalon对象
update(ifInit,x,y) Function update()更新滚动条状态，windowresize，内容高度变化等情况下调用，不能带参数
scrollTo(x,y) Function scrollTo(x,y) 滚动至 x,y
  • scrollbar-document.body带上下小箭头
  • scrollbar-元素带上下小箭头双滚动条
  • scrollbar-带上下小箭头嵌套
  • scrollbar-元素无上下小箭头双滚动条
  • scrollbar-滚动条hover或者滚动时候可见，初始不可见
  • scrollbar-滚动条不可见
