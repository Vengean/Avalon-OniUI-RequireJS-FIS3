switchdropdown

迷你开关型下拉框

基于dropdown开发的迷你开关类型的下拉框

使用说明

名字 默认值 说明
配置参数
width 40 自定义宽度
listWidth 100 自定义下拉列表的宽度
height 60 下拉列表的高度
enable true 组件是否可用
readOnly false 组件是否只读
data [] 下拉列表显示的数据模型
value "" 设置组件的初始值
getTemplate() 模板函数,方便用户自定义模板
参数名/返回值 类型 说明
str String 默认模板
opts Object VM
返回 String 新模板

  • 用select结构配置组件
  • 用options配置组件
  • 用options配置组件并设置duplex
