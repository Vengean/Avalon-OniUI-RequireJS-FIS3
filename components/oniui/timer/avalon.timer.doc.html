timer

时间选择器timer可用于移动端和PC端来选择时间，PC端可以通过mousewheel或者鼠标拖动的方式来操作时间的选择，移动端通过滑动的方式操作时间的选择

使用说明

名字 默认值 说明
配置参数
showItems 9 可视的时间信息条数
showYears false 日期部分是否显示年份
dialog {title: "选择时间", width: 300, showClose: false} 弹层dialog的配置项
width 180 时间选择框的宽度
placeHolder "请选择时间" 配置时间选择器的提示文案
getTimeText(timeDate, placeHolder) 
getTimeText: function(timeDate, placeHolder) {
    var timeText = placeHolder
    if (timeDate) {
        timeText = timeDate.toLocaleString()
    } 
    return timeText
}
选择时间之后的展示格式, timeDate是当前选择时间，如果未选择则为null，选择的话是个Date对象，placeHolder参数是用户配置的时间选择器的提示文案，默认是"请选择时间"
eventType "touch" 设置触发时间选择的事件，配置为"mousewheel"会通过鼠标滚动的方式触发，配置为""或者不配置，组件会根据浏览器支持选择鼠标拖动或者移动的touch事件
mouseWheelSpeed 20 每触发一次mousewheel，元素滚动的距离
useTransform true 是否可以使用css3变换
getDates() avalon.noop 获取日期部分显示的内容,是一个日期信息集合而成的数组，格式为 
[{
    date: "09月16日",
    year: 2014,
    month: 8, 
    day: 16, 
    color: "rgba(51, 51, 51, 1)"
}, {
    date: "09月17日",
    year: 2014,
    month: 8, 
    day: 17, 
    color: "rgba(51, 51, 51, 1)"
},...]
getHours() avalon.noop 获取hour部分显示的内容，是一个hour对象结合而成的数组，格式为： 
[{
    hour: "05", 
    _hour: 5, 
    color: "rgba(51, 51, 51, 1)"
}, {
    hour: "06", 
    _hour: 6, 
    color: "rgba(51, 51, 51, 1)"
},...]
getMinutes() avalon.noop 获取minute部分显示的内容，是一个minute对象结合而成的数组，格式为: 
[{
    minute: "05", 
    _minute: 5, 
    color: "rgba(51, 51, 51, 1)"
}, {
    minute: "10", 
    _minute: 10, 
    color: "rgba(51, 51, 51, 1)"
},...]
getTemplate(str, options) 
function(str, options) {
    return str
}
用于修改模板，传入默认模板与配置对象
onInit(vmodel, options, vmodels) 初始化时触发的回调
外部调用接口
timeDate 获取当前选择时间，返回的是个Date对象,如果没有选择时间，则返回null

  2. 通过鼠标滚动来分别选择dates、hour、minute
