属性绑定(ms-attr-*)

此绑定还是ms-disabled, ms-enabled, ms-readonly, ms-checked, ms-selected的内部实现。

示例 
<!DOCTYPE html>
<html>
    <head>
        <title>ms-attr-*</title>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width">
        <script src="avalon.js"></script>

        <script>
            avalon.define("ms-attr-*", function(vm) {
                vm.aaa = true
                vm.click = function() {
                    vm.aaa = !vm.aaa
                }
                vm.bbb ="@@@"
                vm.ccc = "&&&"
            })

        </script>

    </head>
    <body>
        <form method="get" action="aaa.html" ms-controller="ms-attr-*">
            <input ms-enabled="aaa" name="a1" value="12345"/>
            <input ms-disabled="aaa" name="a2" value="67890"/>
            <input ms-readonly="aaa" name="a3" value="readonly" />

            <input ms-checked="aaa" type="checkbox" value="checkbox" name="a4"/>
            <select name="a5">
                <option>222</option>
                <option ms-selected="aaa">555</option>
            </select>
            <input ms-attr-value="bbb + ccc" name="a6" value="改" />
            <button type="button" ms-click="click">
                点我
            </button>
            <input type="submit" value="提交"/>
        </form>
    </body>
</html>
