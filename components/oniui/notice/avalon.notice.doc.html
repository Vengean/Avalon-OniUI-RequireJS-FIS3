notice

提示组件notice组件用来向用户显示一些提示信息。

使用说明

名字 类型 默认值 说明
配置参数
type String "info" notice类型,可以选择为"success"、"error"或者默认的"info
header String "notice title" notice的标题
title String "notice title" notice的标题
timer Number 0 notice显示之后自动隐藏的定时器，0表示不自动隐藏
hasCloseBtn Boolean true 是否显示关闭按钮，设为false不显示
toggle Boolean false 显示或隐藏notice， true显示，false隐藏
isPlace Boolean true 是否占位，false不占位，true占位，占位时notice显示在自定义的container中，不占位时append到body元素下
isAffix Boolean false 是否吸顶，非占位元素不吸顶，占位元素当滚动距离大于元素距页面顶部距离时吸顶，否则保持原位置
successClass String "oni-notice-info" type为success时的提示类名
errorClass String "oni-notice-danger" type为error时的提示类名
infoClass String "" type为info时的提示类名
zIndex String 'auto' 提示组件的zindex css值
getTemplate(str,options) Function 用于重写模板的函数
参数名/返回值 类型 说明
str String tmpl
options Object opts
返回 String
接口方法与固有属性
content String "" 要显示的内容,可以是DOM对象|String|DOM String
container String "" 显示notice的容器，可以配置为自己想要包裹notice的元素对象或者元素id，id的话必须是"#ID"的格式,默认为绑定组件的元素
onShow Member avalon.noop notice显示之后回调
参数名/返回值 类型 说明
data Object 与此数组相关的数据对象
vmodels Array 位于此组件上方的vmodels组成的数组
onHide Member avalon.noop notice关闭之后回调
参数名/返回值 类型 说明
data Object 与此数组相关的数据对象
vmodels Array 位于此组件上方的vmodels组成的数组
widgetElement String "" accordion容器
  • notice组件概览，具体使用demo及说明请看avalon.notice.ex1...6
  • 使用默认配置的notice组件，可以动态改变其content、header、type
  • 自定义notice的container，设置isPlace可以决定notice是占位还是不占位
  • 当配置notice占位且可吸顶时，notice会在页面滚动到它所在位置时吸顶显示
  • 自定义notice的回调
  • 设置timer使得notice在显示一段时间之后隐藏
  • 自定义notice type的样式类
