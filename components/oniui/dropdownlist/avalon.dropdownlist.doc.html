dropdownlist

下拉提示搜索框

dropdownlist使得可以通过下拉搜索功能减少显示内容，方便用户查找选择

使用说明

名字 类型 默认值 说明
配置参数
searchBox Object null 搜索框对应的VM
getSelected() Function 获取选项值
参数名/返回值 类型 说明
返回 String 选项值
render(data) Function 重新渲染搜索选项列表
参数名/返回值 类型 说明
data Array 选项列表，必传
widgetElement Identifier element 绑定组件的元素引用
placeholder String '' 搜索框的占位符
realTimeData Boolean false 是否动态的从远程获取数据
enable Boolean true 是否禁用组件
getRealTimeData(search,dropdownlist) Function 当需要实时获取搜索数据时设置realTimeData为true，组件就会调用此方法来实时渲染搜索列表
参数名/返回值 类型 说明
search String 搜索内容
dropdownlist Object 组件对应的VM
getTemplate(tmp,opts) Function 模板函数,方便用户自定义模板
参数名/返回值 类型 说明
tmp String 默认模板
opts Object vmodel
返回 String 新模板

dropdownlist组件继承于dropdown组件,因此许多参数请参考dropdown

我们可以通过dropdownVm.value得到当前选中项的value值, 也可以通过dropdownVm.getSelected()得到当前选中项的value值 此外，我们也可能通过dropdownVm.activeIndex得到当前选中项的对应的索引值
