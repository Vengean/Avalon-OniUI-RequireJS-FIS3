accordion

手风琴组件

手风琴组件可以将多个内容组织到多个小面板中，通过点击面板的 小三角(默认)或 标题(需要配置)可以展开或者收缩其内容，使用效果很像 Tab。作为备选，还可以通过将鼠标放置到标题上来展开或者收缩。 此组件能方便我们在有限的区域中放置众多信息。

使用说明

名字 类型 默认值 说明
配置参数
width String '100%' 配置组件宽度(type: Number || Percent)
headerWidth Number 30 组件水平展开时，头部的宽
contentWidth Number 400 组件水平展开时内容的宽
headerAndContentHeight Number 200 组件水平展开时的高度
autoRun Boolean true 告知组件是否自动渲染，设为false时需要手动调用refresh方法进行组件的解析渲染
template String "" 用户自定义template
accordionClass String "" 为accordion容器自定义的class
currentTriggerClass String "oni-state-active" 展开accordion面板时，header添加的class
initIndex Object null 初始展开第几个面板
mode String "caret" 组件展开模式，取值说明："nav"=面板header无小三角图标，"caret"=展开面板有小三角图标，可以定义是点击图标展开面板还是点击header即展开，默认是点击header即展开，当然也可以通过getTemplate自定义模板
multiple Boolean false 是否可以同时打开多个面板
trigger String "oni-accordion-header" 触发展开面板的dom节点对应class，比如mode为caret时想要只通过小图标展开隐藏panel时可以设置为"oni-accordion-trigger"
triggerType String 'click' 触发展开面板的事件类型，可以是：click|mouseenter
direction String "vertical" 组件的展开方向，默认为垂直展开，也可以水平展开("horizontal")
onBeforeSwitch Function avalon.noop 组件面板展开前回调函数
参数名/返回值 类型 说明
index Number 面板序号
header Object 标题区域节点对象
panel Object 面板区域节点对象
返回 {Boolean| Undefined} 若返回false则不展开面板
onSwitch Function avalon.noop 组件面板展开后的回调函数
参数名/返回值 类型 说明
index Number 面板序号
header Object 标题区域节点对象
panel Object 面板区域节点对象
onInit Function avalon.noop 远程更改数据
参数名/返回值 类型 说明
vmodel Object 组件自身vmodel
options Object 组件的配置对象
vmodels Array 组件的祖先vmodel组成的数组链
getTemplate(str,options) Function 方便用户自定义模板
参数名/返回值 类型 说明
str String 默认模板
opts Object vmodel
返回 String 新模板
接口方法与固有属性
rendered Boolean false 组件是否完成渲染,false未完成，true完成
$remove() Function 当组件移出DOM树时,系统自动调用的销毁函数
setData(data) Function 重定义组件配置数据对象
参数名/返回值 类型 说明
data Array 结构如下： 

                [{

                title: "标题1",

                content: "正文1"

                },

                {

                title: "标题2",

                content: "正文2"

                }]
refresh(data) Function 手工刷新组件视图,也可以传递参数data，重渲染组件视图
参数名/返回值 类型 说明
data Array 结构如下： 

                [{

                title: "标题1",

                content: "正文1"

                },

                {

                title: "标题2",

                content: "正文2"

                }]
getCurrentHeader() Function 获得当前展开的accordion标题对象，仅在config.multiple == false时有效
参数名/返回值 类型 说明
返回 ElementObj 标题dom对象的引用
getCurrentPanel() Function 获得当前展开的accordion面板对象，仅在config.multiple == false时有效
参数名/返回值 类型 说明
返回 ElementObj 面板dom对象的引用
getHeader(index) Function 获得指定序号的accordion面板对应的标题节点对象
参数名/返回值 类型 说明
index Number 面板序号
返回 ElementObj 指定序号的标题dom对象的引用
getPanel(index) Function 获得指定序号的accordion面板对应的面板节点对象
参数名/返回值 类型 说明
index Number 面板序号
返回 ElementObj 指定序号的面板dom对象的引用
getLength() Function 获得组件的面板数量
参数名/返回值 类型 说明
返回 Number 手风琴面板个数
getStatus(index) Function 获得指定序号的accordion面板展开(1)/收起(0)状态
参数名/返回值 类型 说明
index Number 指定面板序号(从0开始)
返回 Number 1表示index对应面板展开，0表示收起
switchTo(index) Function 切换accordion面板的展开
参数名/返回值 类型 说明
index Number 指定面板序号(从0开始)
data Array [] 配置accordion组件要展示的数据对象，格式为 

            [

            {title: String, content: String},

            {title: String, content: String},

            {title: String, content: String}

             ]
widgetElement String "" 保存绑定组件元素的引用
currentIndex Unary -1 组件最新展开的面板序号，不可配置
