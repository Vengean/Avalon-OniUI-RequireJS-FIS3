lazyload

延迟加载组件延迟加载组件用于对页面中的图片/文档片段进行延迟加载，让页面在初次渲染时不加载图片/文档片段，当其处于浏览器窗口可视范围内再进行加载，以尽可能提高网页渲染速度。

使用说明

名字 类型 默认值 说明
配置参数
width String "" 懒加载占位宽度，可通过设置元素的data-lazyload-width修改
height String "" 懒加载占位高度，可通过设置元素的data-lazyload-height修改
contentType String "image" 懒加载内容类型："image"-图片 / "DOM"-文档片段
preLoadType String "image" 预加载类型："image"-采用加载中图片 / "text"-采用加载中文字
preLoadSrc Binary path + "/images/loading1.gif" 预加载图片路径（文字内容）：preLoadType为"image"时为图片路径；preLoadType为"text"时为文字内容。也可以设置元素的data-lazyload-preloadsrc，替代默认值
delay Number 500 延迟加载时间（毫秒）。也可以设置元素的data-lazyload-itemdelay，替代默认值
effect String "none" 预加载效果 "none"-无效果 / "fadeIn"-渐入效果 / "slideX"-由左向右滑动 / "slideY"-由上向下滑动，建议在图片加载中使用。也可以设置元素的data-lazyload-itemeffect，替代默认值
easing String "easeInOut" 动画效果的缓动函数
slideDistance Number 300 effect-slide模式的滑动长度。也可以设置元素的data-lazyload-distance，替代默认值
  • 懒加载组件-默认配置
  • 懒加载组件-自定义effect(加载效果)和各自delay(加载延迟)
  • 懒加载组件-自定义加载中图片
  • 懒加载组件-预加载采用文字模式
  • 懒加载组件-懒加载页面元素
  • 懒加载组件-自定义懒加载尺寸
