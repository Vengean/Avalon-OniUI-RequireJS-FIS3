flipswitch

滑动按钮组件

将checkbox表单元素转化成富UI的开关，[不支持ms-duplex，请在onChange回调里面处理类似ms-duplex逻辑] - 在2015.7.13以后已支持 注意：如果指定了ms-duplex，则采用duplex指定的值，接着只采用checked属性为true时情景，最后采用data-flipswitch-cheched以及option.checked，因此可以通过ms-duplex，ms-checked，data-flipswitch-cheched以及option.checked来配置初始值，ms-duplex会进入一个异步scan的逻辑

使用说明

名字 类型 默认值 说明
配置参数
toggle Boolean true 组件是否显示，可以通过设置为false来隐藏组件
onText String " " 选中状态提示文字
size String "normal" 滑动条类型，默认normal，可设置为large,small,mini，以及其他任意组件不自带的名词，可以用来注入自定义class，生成ui-flipswitch-{{size}}添加给flipswitch模板容器
theme String "normal" 主题，normal,success,warning,danger
draggable Boolean false 是否支持拖动切换状态
disabled Boolean false 禁用
checked Boolean false 默认选中状态
animated Boolean true 是否开启切换动画效果
hdir Boolean true 开启、关闭选项排列顺序默认为true，即on-off,false为off-on
onInit Member avalon.noop onInit(vmodel, options, vmodels) 完成初始化之后的回调,call as element's method
_animateArrMaker(from,to) Function _animateArrMaker(from, to) 不支持css3动画效果步长生成器函数，返回一个数组，类似[0,xx,xx,xx,50]
onChange Member avalon.noop onChange(newValue, vmodel) 选中状态发生变化时触发，参数为当前的选中状态及vmodel对象
接口方法与固有属性
toggleStatus() Function toggleStatus 交替改变选中状态
disable() Function disable 禁用组件
enable() Function enable 启用组件
  • 不同大小size，默认设置，选中状态
  • 不同大小size，支持可拖动切换，选中状态
  • 不同大小size，OFF-ON顺序排列，选中状态
  • 不同大小size，disabled状态，选中状态
  • 不同大小size，自定义ON-OFF文本，选中状态
  • 不同大小size，通过接口修改状态，选中状态
  • flipswitch + tooltip
