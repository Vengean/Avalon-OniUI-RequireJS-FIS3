menu

菜单组件

实现扫描dom元素或者设置传参生成级联菜单的组件，注意扫描dom情形下，会销毁原有的dom，且会忽略所有的ol，ul，li元素上原有的绑定

使用说明

名字 类型 默认值 说明
配置参数
active Boolean false 将第几个项目设置为选中，级联情形下，会将设置应用给每一级menu，默认是false，一个都不选中，建议不要通过修改这个值来修改menu的选中状态，而是通过setActiveList接口来做
disabled Boolean false 选中事件，默认mouseenter
dir String "h" 方向，取值v,h，默认h是水平方向， v是竖直方向
onInit Member avalon.noop onInit(vmodel, options, vmodels) 完成初始化之后的回调,call as element's method
menuResetter(vmodel) Function menuResetter(vmodel) 选中某个menu项之后调用的这个restter，默认是把menu重置为不选中
getTemplate(tmpl,opts,tplName) Function getTemplate(tpl, opts, tplName) 定制修改模板接口
onSelect Member avalon.noop onSelect(vmodel, realSelect, _hasSubMenu) this指向选中的menu li元素，realSelect是选中menu项目的数组 
[/n[data, active],/n[data2,active2]/n]
，对应每一级的数据，及每一级的active值，_hasSubMenu表示this元素有无包含子menu
接口方法与固有属性
activate(e,index) Function activate(index)展开菜单索引为index的项目，index置为false,undefined则不会展开任一项目
getActiveList(arr) Function getActiveList() 获取所有选中的menu list
setActiveList(arr) Function setActiveList(activeListArray) 设置级联menu的选项，可以一个数组，也可以使一个数字，或者"2,3,4"这样的字符串
