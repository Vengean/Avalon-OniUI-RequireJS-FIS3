pager

分页组件

分页组件 用于各种列表与表格的下方 。

使用说明

名字 类型 默认值 说明
配置参数
getTitle(a,currentPage,totalPages) Function 获取页码上的title的函数
参数名/返回值 类型 说明
a String|Number a 当前页码的类型，如first, prev, next, last, 1, 2, 3
currentPage Number currentPage 当前页码
totalPages Number totalPages 最大页码
返回 String
perPages Number 10 每页包含多少条目
showPages Number 10 中间部分一共要显示多少页(如果两边出现省略号,即它们之间的页数)
currentPage Number 1 当前选中的页面 (按照人们日常习惯,是从1开始)，它会被高亮
_currentPage Number 1 跳转台中的输入框显示的数字，它默认与currentPage一致
totalItems Number 200 总条目数
totalPages Number 0 总页数,通过Math.ceil(vm.totalItems / vm.perPages)求得
pages Array [] 要显示的页面组成的数字数组，如[1,2,3,4,5,6,7]
nextText String ">" “下一页”分页按钮上显示的文字
prevText String " < " “上一页”分页按钮上显示的文字
ellipseText String "… " 省略的页数用什么文字表示
firstPage Number 0 当前可显示的最小页码，不能小于1
lastPage Number 0 当前可显示的最大页码，不能大于totalPages
alwaysShowNext Boolean false 总是显示向后按钮
alwaysShowPrev Boolean false 总是显示向前按钮
getTemplate(tmpl,opts) Function 用于重写模板的函数
参数名/返回值 类型 说明
tmpl String tmpl
opts Object opts
返回 String
options Array [] 数字数组或字符串数组或对象数组,但都转换为对象数组,每个对象都应包含text,value两个属性, 用于决定每页有多少页(看avalon.pager.ex3.html)
onJump(e,page) Function 页面跳转时触发的函数,如果当前链接处于不可以点状态(oni-state-disabled),是不会触发的
参数名/返回值 类型 说明
e Event e
page Number page 当前页码

pager 组件有一个重要的jumpPage方法，用于决定它的跳转方式。它有两个参数，第一个事件对象，第二个是跳转方式，见源码：

vm.jumpPage = function(event, page) { event.preventDefault() if (page
                    !== vm.currentPage) { switch (page) { case "first ": vm.currentPage = 1
                    break case "last ": vm.currentPage = vm.totalPages break case "next ":
                    vm.currentPage++ if (vm.currentPage > vm.totalPages) { vm.currentPage
                    = vm.totalPages } break case "prev ": vm.currentPage-- if (vm.currentPage
                    < 1) { vm.currentPage = 1 } break default: vm.currentPage = page break
                    } vm.onJump.call(element, event, vm) efficientChangePages(vm.pages, getPages(vm))
                    } }
  • 显示跳转台
  • 指定回调onJump
  • 改变每页显示的数量
  • 指定上一页,下一页的文本
  • 通过左右方向键或滚轮改变页码
  • 总是显示上一页与下一页按钮
  • 多语言支持
