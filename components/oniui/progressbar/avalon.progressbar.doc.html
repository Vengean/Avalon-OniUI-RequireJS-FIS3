progressbar

进度条组件

可以通过接口控制或者随机模拟进度条效果，支持条形，圆形，倒计时等效果功能

使用说明

名字 类型 默认值 说明
配置参数
toggle Boolean true 组件是否显示，可以通过设置为false来隐藏组件
value Boolean false 当前进度值 0 - 100 or false
label Boolean true 是否在进度条上显示进度数字提示
simulate Boolean false 是否模拟进度条效果，默认为否，模拟的时候需要调用触发告知完成，模拟会采用模拟函数及算子进行模拟，取值为int表示动画效果间隔ms数
indeterminate Boolean false 是否不确定当前进度，现在loading效果
countDown Boolean false 倒计时
inTwo Boolean false 是否显示左右两段
circle Boolean false 圆形
circleColor String "#ffffff" 圆形填充色彩，可以配制为从皮肤中提取，只在初始化的时候提取
circleBorderColor String "#dedede" 圆形边框颜色，，可以配制为从皮肤中提取，只在初始化的时候提取
circleBarColor String "#619FE8" 圆形进度条边框颜色，可以配制为从皮肤中提取，只在初始化的时候提取
circleRadius Number 38 圆形的半径，可以配制为从皮肤中提取，只在初始化的时候提取
circleBorderWidth Number 4 圆形的边框宽度，可以配制为从皮肤中提取，只在初始化的时候提取
success Boolean false 是否完成，进度为100时或者外部将success置为true，用于打断模拟效果
onInit Member avalon.noop onInit(vmodel, options, vmodels) 完成初始化之后的回调,call as element's method
simulater(i,vmodel) Function simulater(value, vmodel) 模拟进度进行效果函数，参数为当前进度和vmodel，默认return value + 5 * Math.random() >> 0
getTemplate(tmpl,opts) Function getTemplate(tmp, opts) 用于修改模板的接口，默认不做修改
onChange Member avalon.noop onChange(value) value发生变化回调，this指向vmodel
onComplete Member avalon.noop onComplete() 完成回调，默认空函数，this指向vmodel
接口方法与固有属性
start() Function start() 开始进度推进，该接口适用于模拟进度条
end(value) Function end(value) 结束进度推进，该接口适用于模拟进度条，value为100表示结束，failed表示失败，undefine等于pause，其他则终止于value，并在label上显示
reset(value) Function reset(value) 重置设置项，参数可选，为需要重设的值
progress(value) Function progress(value) 设置value值，其实也可以直接设置vmodel.value
  • 外部设置进度值
  • 模拟进度条，带label
  • 外部设置进度值，带label
  • 模拟进度条，带label，进度indeterminate，接口调用
  • 占比进度条，带label，左右两段，接口调用
  • 模拟进度条倒计时
  • 圆形进度条，外部设置进度值，带label
  • 圆形进度条，外部设置进度值，带label，倒计时
