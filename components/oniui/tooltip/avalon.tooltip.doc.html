tooltip

气泡组件

给一个元素或者给元素里满足配置条件的系列元素添加一个富UI及交互的气泡提示框

使用说明

名字 类型 默认值 说明
配置参数
toggle Boolean false 组件是否显示，可以通过设置为false来隐藏组件，设置为true来显示【在原来的位置展示原来的内容，如果需要改变位置、内容，请使用showBy】
collision String "none" 溢出检测，当被定位元素在某些方向上溢出窗口，则移动它到另一个位置。与 my 和 at 选项相似，该选项会接受一个单一的值或一对 horizontal/vertical 值。例如：flip、fit、fit flip、fit none。/nflip：翻转元素到目标的相对一边，再次运行 collision 检测一遍查看元素是否适合。无论哪一边允许更多的元素可见，则使用那一边。/nfit：把元素从窗口的边缘移开。/nflipfit：首先应用 flip 逻辑，把元素放置在允许更多元素可见的那一边。然后应用 fit 逻辑，确保尽可能多的元素可见。/nnone: 不检测
event String "mouseenter" 显示tooltip的事件，默认hover的时候显示tooltip，为false的时候就不绑定事件，如果后面设置了自动隐藏，则mouseenter对应的是mouseleave,focus对应的是blur，进行自动隐藏事件侦听，使用代理的时候，目测不支持focus,blur，event可以配置为空，则不会添加事件侦听
content Unary void 0 tooltip显示内容，默认去获取element的title属性
container Element void 0 把tooltip元素append到container指定的这个元素内，可以是一个函数，用以返回一个元素
width String "auto" tip宽度，默认是auto
height String "auto" tip高度，默认是auto
arrow Boolean true 是否显示尖角图标，默认为true
autohide Boolean true 元素hoverout之后，是否自动隐藏tooltip，默认true
delegate Boolean false 元素是否只作为一个代理元素，这样适合对元素内多个子元素进行tooltip绑定
disabled Boolean false 禁用
track Boolean false tooltip是否跟随鼠标，默认否
animated Boolean true 是否开启显示隐藏切换动画效果
position String "rt" tooltip相对于element的位置，like: rt,rb,rc...
positionMy Boolean false tooltip元素的定位点，like: left top+11
positionAt Boolean false element元素的定位点，like: left top+11,positionAt && positionMy时候忽略position设置
hiddenDelay Number 16 tooltip自动隐藏时间，单位ms
zIndex Number maxZIndex tooltip的z-index值，默认是body直接子元素中的最大z-index值
onInit Member avalon.noop onInit(vmodel, options, vmodels) 完成初始化之后的回调,call as element's method
contentGetter(elem) Function contentGetter() 获取内容接口，讲srcElement作为参数传递过来，默认是返回a标签的title，如果该函数返回为空，那么则不会显示tooltip
_animateArrMaker(from,to) Function _animateArrMaker(from, to) 不支持css3动画效果步长生成器函数，返回一个数组，类似[0,xx,xx,xx,50]
getTemplate(tmpl,opts) Function getTemplate(tpl, opts) 定制修改模板接口
接口方法与固有属性
show(elem) Function show(elem) 用这个方法来刷新tooltip的位置
hide(e) Function hide($event) 隐藏tooltip，参数是$event，可缺省
showBy(obj,content) Function showBy($event, content) 参数满足 {target: elem}这样，或者是一个elem元素亦可，tooltip会按照elem定位，并作为参数传递给contentGetter，如果指定content，则忽略contentGetter的返回，直接显示content内容
参数名/返回值 类型 说明
obj $event 定位参照物，可以是一个事件，也可以是一个元素，如果未提供有效的元素或者事件，则采用上一次定位的元素或者事件来定位
content content 用来填充tooltip的内容
appendTo(ele) Function 将toolTip元素注入到指定的元素内，请在调用appendTo之后再调用showBy
参数名/返回值 类型 说明
ele 目标元素
  • input元素不绑定事件，人工触发
  • input元素hover显示tip-父容器做代理
  • 不使用代理，各个方向绑定，改变位置，点击展示
  • 箭头位置自动调整，点击展示
  • input元素获取焦点显示tip，不展示箭头
  • 鼠标跟随
  • input元素获取焦点显示tip
  • 显示tooltip时动态获取最大z-index
