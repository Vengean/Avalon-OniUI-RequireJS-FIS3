checkboxlist

Name 复选框列表

通过checkboxlist可以方便的实现选框的全选、全不选，并可通过简单配置进行选中操作的回调处理，也可以通过多种方式来提供渲染选项视图所需要的数据

使用说明

名字 类型 默认值 说明
配置参数
data Array [] 所有选项值的集合，通过此数据来渲染初始视图。可以在组件初始化之前配置data，也可以在异步取得数据之后在配置。当同时配置了data、fetch且在绑定元素内部显示设置了要渲染的checkbox列表，则优先级顺序是：data>fetch>sub elements
all Boolean false 默认不选中所有选项
alltext String "全部" 显示"全部"按钮，方便进行全选或者全不选操作,不需要全选操作的话可以设置alltext为""
type String "" 内置type为week时的data，用户只需配置type为week即可显示周一到周日的选项
fetch String "" 通过配置fetch来获得要显示的数据，数据格式必须如下所示： 
             [
                { text : '文字1' , value : 'w1' } ,
                { text : '文字2' , value : 'w2' } ,
                { text : '文字3' , value : 'w3' } ,
                { text : '文字4' , value : 'w4' }
             ]
onSelect Function avalon.noop 组件面板展开后的回调函数
参数名/返回值 类型 说明
data Array checkboxlist的选项集合
checkStatus Boolean 选中或者未选中的状态
target ElementObj 触发事件的dom对象的引用
vertical Boolean true 如果希望选框水平排列则设置vertical为false，默认垂直排列

  • checkboxlist组件默认提供了type为week时的data
  • 配置checkboxlist-fetch获取用户定义的所有选项值
  • 配置onselect回调
